小白 OpenClaw 新手指南

零基礎也能上手的 AI 私人助手 —— 從安裝到日常使用，一篇搞定。

OpenClaw 是什麼？

OpenClaw 是一個開源、本地運行的 AI 私人助手框架，由 Peter Steinberger 開發。

和普通聊天機器人不同，OpenClaw 能真正幫你做事 —— 管郵件、排日程、抓網頁數據、執行終端命令，甚至控制智能家居，而你的數據始終留在自己電腦上。

簡單來說：你給它一個任務，它自己去完成，而不只是回答你的問題。

為什麼選 OpenClaw？

🔒

隱私優先

所有數據本地運行，不經過第三方伺服器。你的聊天記錄、檔案、密碼全都留在你自己的電腦上。

🤖

真正的 AI 代理

它能上網搜索、讀寫檔案、執行命令、發消息，像一個真正的助手一樣主動行動。不單單只是

💬

多平台接入

透過 WhatsApp、Telegram、Discord、以及 Slack 等平台對接對話。

🧠

持久記憶

它會記住你的偏好和習慣，用得越久越懂你。不像普通 AI 每次聊天都從零開始。

事前準備

  • 電腦：macOS、Windows 或 Linux 都可以（推薦 8GB 記憶體以上）
  • 網路：需要連網下載安裝包和呼叫 AI 模型
  • AI API Key：註冊 Anthropic（Claude） 或 OpenAI（GPT） 取得 API Key
  • 通訊軟體（可選）：Telegram 或 WhatsApp 帳號，用來遠程和 AI 對話

三步安裝 OpenClaw

第一步：安裝

macOS / Linux —— 打開終端（Terminal），貼上以下命令：

curl -fsSL https://openclaw.ai/install.sh | bash

Windows —— 打開 PowerShell，貼上以下命令：

iwr -useb https://openclaw.ai/install.ps1 | iex

也可以透過 npm 安裝：

npm i -g openclaw

第二步：初始化設定

安裝完成後，執行設定精靈：

openclaw init

精靈會引導你：

  • 選擇 AI 模型（推薦 Claude Sonnet —— 性價比最高）
  • 輸入你的 API Key
  • 連接通訊頻道（Telegram、WhatsApp 等）
  • 設定基本偏好（語言、時區等）

第三步：開始對話

設定完成後，直接在終端輸入：

openclaw chat

或者打開你綁定的 Telegram / WhatsApp，直接給 OpenClaw 發消息就行了！

它是怎麼運作的？（架構簡述）

OpenClaw 採用四層架構，你不需要完全理解，但了解基本原理有助於排查問題：

  1. 通訊層（Channel） —— Telegram、WhatsApp 等訊息入口
  2. 閘道層（Gateway） —— Node.js 中樞，管理狀態和任務調度
  3. 大腦層（LLM Brain） —— Claude、GPT 或本地模型，負責思考和決策
  4. 技能層（Skills） —— TypeScript 模組，定義具體能力（瀏覽器操作、檔案管理等）

還有一個特殊功能叫 Heartbeat（心跳）：它是一個後台計時器，讓 OpenClaw 可以定期自動執行任務，比如每天早上給你發新聞摘要。

安全注意事項（非常重要）

OpenClaw 的一位核心維護者曾警告：「如果你不懂命令列，這個專案對你來說太危險了。」 給 AI 終端存取權限相當於給了它 「上帝模式」，所以安全措施必須到位。

新手必做的安全清單：

  • 建立獨立使用者：建立一個專門的 ai-agent 系統使用者，限制它的檔案存取範圍
  • 使用 Docker 容器：把 OpenClaw 跑在容器裡，限制它能看到的檔案和目錄
  • 設定 API 花費上限：在 API 平台設定每日花費上限（建議 $5-$10），避免意外高額帳單
  • 白名單機制：只允許你自己的 Telegram/WhatsApp ID 發送指令
  • 審批機制：對危險命令（rm、sudo、curl）啟用人工確認
  • 不要給銀行密碼：任何涉及金融操作的場景，永遠不要讓 AI 直接接觸你的銀行憑據

費用參考

  • OpenClaw 本身：完全免費開源
  • AI 模型 API：按用量計費，大約每 100 個任務 $0.50 – $2.00（使用 Claude Sonnet）
  • 硬體：推薦使用 Mac Mini 或 Intel NUC 作為 24/7 運行的專用設備

一般個人使用者，每月 AI API 花費大概在 $5 – $20 左右。

給小白的 5 個建議

  1. 先唯讀不寫：剛開始只給 OpenClaw 讀取權限，觀察 48 小時確認它的行為後再開放寫入/執行權限
  2. 從簡單任務開始：先讓它幫你查天氣、總結文章，別一上來就讓它管你的信箱
  3. 善用 Telegram：Telegram 是最成熟的通訊頻道，設定最簡單，推薦新手首選
  4. 定期檢查日誌：OpenClaw 會記錄所有執行過的命令，養成定期查看的習慣
  5. 加入社群：遇到問題去 GitHub 或 Discord 社群提問，社群非常活躍

總結

OpenClaw 是目前最強大的開源 AI 代理框架之一。它把 AI 從「只能聊天」升級成了「能真正幹活的私人助手」。

雖然上手需要一點點技術基礎，但只要按照本指南一步步來，即使是完全的小白也能把它跑起來。

記住：先求穩，再求快。安全永遠是第一位的。
