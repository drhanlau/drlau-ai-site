小白 OpenClaw 新手指南

零基础也能上手的 AI 私人助手 —— 从安装到日常使用，一篇搞定。

OpenClaw 是什么？

OpenClaw 是一个开源、本地运行的 AI 私人助手框架，由 Peter Steinberger 开发。 和普通聊天机器人不同，OpenClaw 能真正帮你做事 —— 管邮件、排日程、抓网页数据、执行终端命令，甚至控制智能家居，而你的数据始终留在自己电脑上。

简单来说：你给它一个任务，它自己去完成，而不只是回答你的问题。

为什么选 OpenClaw？

🔒

隐私优先

所有数据本地运行，不经过第三方服务器。你的聊天记录、文件、密码全都留在你自己的电脑上。

🤖

真正的 AI 代理

不只是回答问题 —— 它能上网搜索、读写文件、执行命令、发消息，像一个真正的助手一样主动行动。

💬

多平台接入

通过 WhatsApp、Telegram、Discord、Slack、Signal、iMessage 等 50+ 平台和它对话。

🧠

持久记忆

它会记住你的偏好和习惯，用得越久越懂你。不像普通 AI 每次聊天都从零开始。

开始之前你需要准备

  • 电脑：macOS、Windows 或 Linux 都可以（推荐 8GB 内存以上）
  • 网络：需要联网下载安装包和调用 AI 模型
  • AI API Key：注册 Anthropic（Claude） 或 OpenAI（GPT） 获取 API Key
  • 通讯软件（可选）：Telegram 或 WhatsApp 账号，用来远程和 AI 对话

三步安装 OpenClaw

第一步：安装

macOS / Linux —— 打开终端（Terminal），粘贴以下命令：

curl -fsSL https://openclaw.ai/install.sh | bash

Windows —— 打开 PowerShell，粘贴以下命令：

iwr -useb https://openclaw.ai/install.ps1 | iex

也可以通过 npm 安装：

npm i -g openclaw

第二步：初始化设置

安装完成后，运行设置向导：

openclaw init

向导会引导你：

  • 选择 AI 模型（推荐 Claude Sonnet —— 性价比最高）
  • 输入你的 API Key
  • 连接通讯渠道（Telegram、WhatsApp 等）
  • 设置基本偏好（语言、时区等）

第三步：开始对话

设置完成后，直接在终端输入：

openclaw chat

或者打开你绑定的 Telegram / WhatsApp，直接给 OpenClaw 发消息就行了！

它是怎么工作的？（架构简述）

OpenClaw 采用四层架构，你不需要完全理解，但了解基本原理有助于排查问题：

  1. 通讯层（Channel） —— Telegram、WhatsApp 等消息入口
  2. 网关层（Gateway） —— Node.js 中枢，管理状态和任务调度
  3. 大脑层（LLM Brain） —— Claude、GPT 或本地模型，负责思考和决策
  4. 技能层（Skills） —— TypeScript 模块，定义具体能力（浏览器操作、文件管理等）

还有一个特殊功能叫 Heartbeat（心跳）：它是一个后台定时器，让 OpenClaw 可以定期自动执行任务，比如每天早上给你发新闻摘要。

安全注意事项（非常重要）

OpenClaw 的一位核心维护者曾警告："如果你不懂命令行，这个项目对你来说太危险了。" 给 AI 终端访问权限相当于给了它 "上帝模式"，所以安全措施必须到位。

新手必做的安全清单：

  • 创建独立用户：创建一个专门的 ai-agent 系统用户，限制它的文件访问范围
  • 使用 Docker 容器：把 OpenClaw 跑在容器里，限制它能看到的文件和目录
  • 设置 API 花费上限：在 API 平台设置每日花费上限（建议 $5-$10），避免意外高额账单
  • 白名单机制：只允许你自己的 Telegram/WhatsApp ID 发送指令
  • 审批机制：对危险命令（rm、sudo、curl）启用人工确认
  • 不要给银行密码：任何涉及金融操作的场景，永远不要让 AI 直接接触你的银行凭据

费用参考

  • OpenClaw 本身：完全免费开源
  • AI 模型 API：按用量计费，大约每 100 个任务 $0.50 – $2.00（使用 Claude Sonnet）
  • 硬件：推荐使用 Mac Mini 或 Intel NUC 作为 24/7 运行的专用设备

一般个人用户，每月 AI API 花费大概在 $5 – $20 左右。

给小白的 5 个建议

  1. 先只读不写：刚开始只给 OpenClaw 读取权限，观察 48 小时确认它的行为后再开放写入/执行权限
  2. 从简单任务开始：先让它帮你查天气、总结文章，别一上来就让它管你的邮箱
  3. 善用 Telegram：Telegram 是最成熟的通讯渠道，设置最简单，推荐新手首选
  4. 定期检查日志：OpenClaw 会记录所有执行过的命令，养成定期查看的习惯
  5. 加入社区：遇到问题去 GitHub 或 Discord 社区提问，社区非常活跃

总结

OpenClaw 是目前最强大的开源 AI 代理框架之一。它把 AI 从"只能聊天"升级成了"能真正干活的私人助手"。 虽然上手需要一点点技术基础，但只要按照本指南一步步来，即使是完全的小白也能把它跑起来。

记住：先求稳，再求快。安全永远是第一位的。
